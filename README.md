# Jin Sun Academic Homepage

This is a minimal static academic homepage for GitHub Pages.

## Template Choice

The layout follows the spirit of `mavroudisv/plain-academic`: a plain static
academic website with a narrow profile/navigation column and a readable main
content column. The Projects section is intentionally omitted.

## Local Preview

Open `index.html` in a browser. Because the site is fully static, no local
server is required. The site uses separate pages for `Home`, `Publications`,
`Presentations`, and `CV`.

## Publish on GitHub Pages

1. Create a repository named `username.github.io`, or use an existing project
   repository.
2. Add the HTML files, `styles.css`, `.nojekyll`, and the `assets` directory.
3. In GitHub, open `Settings -> Pages`.
4. Set the source to the `main` branch and root directory.
5. Save, then visit the Pages URL after GitHub finishes deployment.

## Updating Content

- Add new papers in the `Preprints` list in `index.html`.
- Add new papers in `publications.html`.
- Add talks in `presentations.html`.
- Update education and honors in `cv.html`.
- Update the footer date after a visible content change.
